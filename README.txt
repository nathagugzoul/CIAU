ICU WebApp Officielle

Instructions de déploiement sur GitHub Pages :
1. Décompressez ce zip.
2. Uploadez l'intégralité du contenu dans votre dépôt 'ciau'.
3. Dans Settings > Pages, sélectionnez la branche 'main' (ou 'master') et le dossier root.
4. Votre site sera accessible à https://<votre-pseudo>.github.io/ciau/
